## Coursera Project
Human Activity recognition using Smartphones.
Data Set was downloaded from UCI Machine learning repository.
#Data Sets Used:
  a)features.txt: Has list of features which are used to identify the values in dataset.
  b) activity_labels: Links the class labels with their activity name.
  c)Train set
  d) Test set
  e) Subject_train, subject_test:  Each row identifies the subject who performed the activity for each window sample. Its range is from 1 to 30. 

run_analysis.R loads and reads the data sets and tidy the data. Particular feature extraction can also be done.
## Requirements
libraries: 'data.table' and 'dplyr'
