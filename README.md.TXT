# Coursera Project - Getting and Cleaning Data


This repository contains Coursera Project "Human activity detection using Smartphones Data Set(HAR). The data set was 
provided by UCI Machine Learning website.
You can find the data set in following website.
"https://d396qusza40orc.cloudfront.net/getdata%2Fprojectfiles%2FUCI%20HAR%20Dataset.zip"

Repository Contents and Information
This repository contains,
1. run_analysis.R Script which, 
  a) Loads and reads the training and test data set. 
  b) Merges the both files
  c) Extacts Standard deviation and Mean() measurements.
  d)Appropriately labels the data set with descriptive variable name.
  e) creates tidy data set and saves the data to text file "tidy_summary.txt"
  2) Data set downloaded from UCI website
  3) "tidy_summary.txt" which has cleaned and processed data
  4) codebook
